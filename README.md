 Damped Harmonic Oscillator Simulation
Overview
This project simulates a damped harmonic oscillator using the Euler–Cromer numerical method in C.
It calculates the time evolution of position, velocity, and energy of the oscillator.
Features
* Numerical simulation of oscillator motion
* Includes damping effect
* Tracks energy decay over time
Output
The program prints time, position, velocity, and energy which can be plotted for analysis.
